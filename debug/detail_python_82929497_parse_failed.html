## 🏷️ 应用开发工程师（Agent 方向） · 研发部| 项目 | 详情 ||------|------|| 岗位名称 | 应用开发工程师（Agent 方向） || 薪资范围 | 20-30k || 社保福利 | 六险一金 || 工作地点 | 上海 || 街道地址 | 上海市浦东新区张江高科技园区科技路 1 号 || 详细地址 | A 座 10 层 1001 室 || 学历要求 | 本科及以上 || 经验要求 | 3 年以上相关开发经验，有行业内 Agent / 大模型应用落地经验优先 |---## 📋 你将负责的事**大模型 Agent 架构设计与开发**负责基于主流大模型（如 GPT、Claude、通义千问等）的 Agent 应用架构设计，独立完成从需求分析到上线交付的全流程。衡量标准：Agent 能稳定运行、响应延迟可控、用户交互自然。**Agent 工作流编排与工具集成**设计并实现 Agent 的推理链路（ReAct / Plan-and-Execute / Multi-Agent 等模式），集成外部 API、数据库、知识库等工具，使 Agent 具备真实业务执行能力。衡量标准：工具调用成功率 ≥ 95%，复杂任务完成度可量化。**TypeScript 工程化落地**使用 TypeScript 编写高质量、可维护的 Agent 应用代码，包括核心逻辑层、API 服务层、前端交互层。衡量标准：代码通过 Code Review，单元测试覆盖率达标，线上无明显缺陷。**Prompt 工程与效果调优**持续优化 Agent 的 System Prompt、Few-shot 示例和工具定义，通过 A/B 测试和数据回传迭代 Prompt 策略。衡量标准：Agent 输出质量评分稳步提升，用户满意度改善。**技术调研与方案选型**跟踪 Agent 开发领域的最新框架（LangChain、AutoGen、CrewAI 等）和最佳实践，为团队引入合适的技术方案。衡量标准：技术方案文档清晰，PoC 验证可复现。**协作与知识沉淀**与产品、算法、前端团队紧密协作，将 Agent 能力嵌入到实际业务场景；输出技术文档和内部分享。衡量标准：文档完整度、跨团队协作顺畅度。---## 🎯 我们期待你**必须具备 — 3 年以上 TypeScript 全栈开发经验**熟练掌握 TypeScript 语言特性及生态（Node.js / Express / NestJS / React 等），能独立完成从后端 API 到前端交互的全链路开发。**必须具备 — 大模型 Agent 实战经验**有过基于大语言模型（LLM）构建 Agent 应用的真实项目经验，理解 Agent 的核心概念（工具调用、记忆管理、规划推理、多 Agent 协作），能独立搭建可工作的 Agent 原型。**必须具备 — API 集成与数据流处理能力**有丰富的 RESTful / GraphQL API 集成经验，熟悉向量数据库（如 Milvus、Pinecone）或传统数据库，能处理 Agent 运行中的数据流转和状态管理。**必须具备 — 本科及以上学历，计算机 / 软件工程 / 人工智能相关专业**具备扎实的计算机基础（数据结构、算法、网络），有良好的编码习惯和工程素养。---## ✨ 有这些经历的候选人优先· **Agent 框架深度使用**：熟悉 LangChain、AutoGen、CrewAI、Semantic Kernel 等至少一个框架，并在生产环境中落地过· **RAG / 知识库经验**：有检索增强生成（RAG）系统搭建经验，了解 Embedding、向量检索、文档分块策略· **多 Agent 协作系统**：设计或参与过多 Agent 协同工作的系统，理解 Agent 间通信、任务分配、冲突解决机制· **性能调优与可观测性**：对 Agent 的响应延迟、Token 消耗、工具调用失败率有监控和优化经验· **AI 原生产品思维**：不局限于"把 LLM 当接口调"，能从用户场景出发设计 Agent 交互流程和边界处理· **开源贡献或技术社区活跃**：在 GitHub 有 AI / Agent 相关开源项目，或在技术社区有 Agent 实践分享
